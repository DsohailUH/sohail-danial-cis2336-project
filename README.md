# ArtConnect  
A simple web-based art showcase platform designed to highlight local artwork, promote community art events, and allow artists to submit their work. This project was created as part of an academic assignment and demonstrates core web development skills including HTML, CSS, JavaScript, form validation, multimedia embedding, and interactive UI components.

---

## 📌 Features

### **Homepage**
- Website title and tagline  
- Featured artwork  
- Featured event  
- Embedded multimedia (YouTube video)  
- Developer contact information  

### **Gallery Page**
- Display of multiple artworks  
- JavaScript-powered search bar for filtering artworks  
- Responsive grid layout  

### **Events Page**
- List of upcoming art events  
- Expand/collapse event details using JavaScript  
- Clean card-based layout  

### **Artist Submission Page**
- Form for artists to submit artwork  
- Full JavaScript validation (name, email, title, description, category, price, image)  
- Success message on valid submission  

### **FAQ Page**
- Click-to-expand questions using JavaScript  

### **References Page**
- Image sources  
- Multimedia source  
- External resources  
- AI prompts used  
- Developer information  

---

## 🛠️ Technologies Used

- **HTML5** — Structure and content  
- **CSS3** — Styling and layout  
- **JavaScript (Vanilla)** — Interactivity and form validation  
- **GitHub Pages** — Hosting the project
